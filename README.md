# Car-sharing

One-page website for the car rental company. Created in HTML and CSS. 

## Instalation

Clone the source repository from Github. On the command line enter:

```
git clone
```

Project setup:

```
npm install
```

Project run:

```
npm start
```

## Built With:

* CSS
* HTML
